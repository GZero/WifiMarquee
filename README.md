# WifiMarquee
An ESP8266 based scrolling marquee that can be controlled through a web-based wifi interface.

# Requirements:

  MicroPython 1.10 or later

# Wiring:

 Wemos D1 Mini     MAX7219 LED Matrix
 
    5V         <-->     VCC
    
    GND        <-->     GND
    
    D7 (MOSI)  <-->     DIN
    
    D8         <-->     CS
    
    D5 (SCK)   <-->     CLK
